flom_conns_s::array use cases:
- flom_conns_add() add a new connection (must be postponed)
- flom_conns_clean() remove useless connections (must be postponed)
- flom_conns_close_fd() close the associated file descriptor and mark a
                        connection "useless" (must be postponed)
- flom_conns_free() remove all allocated memory (clean-up only)
- flom_conns_get_cd() read only
- flom_conns_get_fd() read only
- flom_conns_get_fds() read only, generates a new poll_array (it must be
                       synchronized with array state)
- flom_conns_get_gmpc() read only
- flom_conns_get_msg() read only
- flom_conns_import() add an existing connection (must be postponed)
- flom_conns_init() create a new object (set-up only)
- flom_conns_set_events() set events on poll_array (it must be called after
                          _get_fds() and must be synchronized with array state)
- flom_conns_trace() read only
- flom_conns_trns_fd() remove a connection to be transferred to a different
                       connections object (must be postponed)
